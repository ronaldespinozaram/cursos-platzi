Agregar nuevos colaboradores al proyecto 
1. en el proyecto original hay que ir al setting del repositorio.
setting / Manage access
    invite a colaborator

2. la invitacion le llega al colaborador por un correo y este
debe aceptar la invitacion.

3. luego clonar el proyecto.
    hacer cambios al proyecto.
        git commit -am "asdf"
    
    hacer un push al servidor con los cambios realizados.

>> COLABORADORES REMOTOS
1. buscamos el proyecto en github
2. hacemos watch start y FORK.
3. Luego de hacer el fork en nuestro terminal local clonamos el proyecto
4. hacemos los cambios.
